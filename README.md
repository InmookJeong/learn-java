### 1. learn-java Repository의 목적
- Java 프로그래밍 언어의 구조와 제공되는 기능에 대한 개념 정리
- 개발하는 과정에서 필요한 기능을 Google에 검색한 후 검증하지 말고 내 Repository 안에서 찾아 적용할 수 있도록 정리
- 먼 미래에 '누군가'를 교육할 때 교육자료로 사용할 수 있도록 정리 

### 2. 폴더 구조
(1) document
-  Java 프로그래밍 언어의 구조와 개념에 대해 정리한 폴더입니다.
- 개념 정리에 필요한 예제도 같이 작성됩니다.

(2) source
- `document`에서 작성한 예제 소스코드를 관리하기 위한 폴더입니다.
- Java Project를 만들어 소스코드를 관리하며, 예제 소스코드를 상세하게 볼 경우 `source` 폴더 내에서 확인할 수 있습니다.

### 3. 목차
> 목차는 `document` 폴더를 기준으로 작성됩니다.
